math
----

Computes math operations

usage

.. code-block:: none

    math [operation]

example

.. code-block:: none

    math 1 + 1

output

.. code-block:: none

    1 + 1
    => 2.0


rageface
--------

Display a rageface according to given [face] argument.

usage

.. code-block:: none

    rageface [face]

list available faces

.. code-block:: none

    rageface help
